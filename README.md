# Nuxt Shadcn template

Nuxt 4 template repository to bootstrap new projects.
Including Shadcn component library.

## Stack
- Nuxt 4
- Vue 3 + Vue Router 5
- Tailwind CSS v4
- Pinia
- @nuxtjs/i18n
- @nuxtjs/color-mode
- @nuxt/image, @nuxt/icon, @nuxt/fonts
- TypeScript + ESLint
- pnpm

## Project structure
- `app/` Nuxt app directory
- `app/pages/` routes
- `app/layouts/` layouts
- `app/assets/css/main.css` Tailwind entry
- `public/` static assets

## Scripts
- `pnpm dev` start dev server
- `pnpm build` build for production
- `pnpm generate` static generation
- `pnpm preview` preview build
- `pnpm lint` run ESLint
- `pnpm lint:fix` auto-fix lint issues
